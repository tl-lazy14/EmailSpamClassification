Email Spam Classification

1, Giới thiệu.
	Đây là một chương trình sử dụng phương pháp học máy Multinomial naive Bayes cho bài toán phân loại thư điện tử và đánh giá độ hiểu quả mô hình trong bài toán này.

2, Cài đặt.
	- Cài đặt ngôn ngữ Python và Jupyter về máy tính.
	- Tải và giải nén file source code về máy.
    - Chạy lệnh 'pip install -r requirements.txt' để cài đặt các thư viện cần thiết

3, Sử dụng.
    - Đi đến thư mục chính trên terminal
    - Chạy lệnh streamlit run app.py 
    - Sử dụng trên giao diện web localhost, nhập tiêu đề và nội dung email để kiếm tra xem nó có phải email spam hay không
4, Công nghệ sử dụng.
	- Python 3.11.3
	- pandas 2.0.2
	- numpy 1.25.0
	- sklearn
	- matplotlib 3.7.1
    - seaborn
    - nltk
    - wordcloud
    - scipy
    - beautifulsoup4
    - streamlit
